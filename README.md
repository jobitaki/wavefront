# wavefront

Dataflow visualizer available in two versions:

## Web Version
Open `index.html` in your browser. Example dot graph and fire.log are available in the examples directory.

## PyQt Desktop Version
Run the desktop application using Python:
```bash
pip install -r requirements.txt
python3 wavefront_qt.py
```

See [README_PYQT.md](README_PYQT.md) for detailed installation and usage instructions.
